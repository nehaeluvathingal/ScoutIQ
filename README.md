\# 🚀 ScoutIQ



\### \*The AI That Scouts Opportunities Before You Do\*



ScoutIQ is an AI-powered multi-agent career assistant that helps job seekers discover relevant opportunities, verify employer trust, track applications, and automatically monitor recruiter emails—all from one intelligent workspace.



Built with \*\*Google ADK\*\*, \*\*FastAPI\*\*, \*\*Streamlit\*\*, \*\*SQLite\*\*, \*\*Gemini\*\*, and \*\*Model Context Protocol (MCP)\*\*, ScoutIQ combines multiple specialized AI agents to simplify the job search process.



\---



\# ✨ Features



\- 📄 AI Resume Analysis

\- 🎯 Personalized Job Discovery

\- 🛡️ Company Trust \& Scam Detection

\- 📋 Application Tracking

\- 📧 Recruiter Email Monitoring

\- 🤖 Multi-Agent Architecture using Google ADK

\- 🔗 MCP Integration

\- ⚡ SHA-256 Resume Caching

\- 🔍 Hybrid Email Classification (Rule-Based + Gemini)



\---



\# 🏗️ Architecture



The platform consists of multiple AI agents working together:



\- Resume Analyzer Agent

\- Job Discovery Agent

\- Company Verification Agent

\- Application Tracker Agent

\- Email Monitoring Agent



These agents communicate through Google ADK and MCP to provide an intelligent, end-to-end job search experience.

\# ⚙️ Installation



Clone the repository



```bash

git clone https://github.com/nehaeluvathingal/ScoutIQ.git

```



Move into the project folder



```bash

cd ScoutIQ

```



Install dependencies



```bash

pip install -r requirements.txt

```



\---



\# ▶️ Running the Project



Start the backend



```bash

python main.py

```



Run the frontend



```bash

streamlit run app.py

```
